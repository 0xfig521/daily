# Claw 资讯 🦀

> 2026-04-17 每日资讯

## openclaw 2026.4.15 发布：Claude Opus 4.7 和 Gemini TTS 支持

### 主要更新

**Anthropic/模型更新**
- Anthropic 模型默认选择升级为 Claude Opus 4.7
- 新增 `opus` 别名和 Claude CLI 默认值
- 捆绑的图像理解功能升级至 Claude Opus 4.7

**Google/TTS**
- 为捆绑的 `google` 插件添加 Gemini 文字转语音支持
- 包括提供商注册、语音选择、WAV 回复输出、PCM 电话输出以及设置/文档指导

**控制 UI/概览**
- 新增模型认证状态卡片，显示 OAuth 令牌健康状况和提供商速率限制压力
- 当 OAuth 令牌即将过期或已过期时发出提醒
- 新的 `models.authStatus` 网关方法 stripping 凭证并缓存 60 秒

**内存/LanceDB**
- 为 `memory-lancedb` 添加云存储支持
- 持久化内存索引现在可以在远程对象存储而非仅本地磁盘运行

**GitHub Copilot/内存搜索**
- 新增 GitHub Copilot 嵌入提供商用于内存搜索
- 插件可复用传输层同时支持远程覆盖、令牌刷新和更安全的负载验证

**代理/本地模型**
- 新增实验性配置 `agents.defaults.experimental.localModelLean: true`
- 丢弃 `browser`、`cron`、`message` 等重量级默认工具
- 为较弱的本地模型设置减少提示大小

### 修复内容

- 网关/工具：修复客户端工具名称与内置工具冲突时的安全信任问题
- 代理/回放恢复：正确分类提供商 401 错误为 replay-invalid
- Docker/构建：修复 pnpm v10+ 虚拟存储布局导致镜像构建失败问题
- Matrix/E2EE：改进无密码令牌认证机器人的启动引导
- Dreaming/内存核心：Dreaming 阶段块现在默认存储在独立文件而非内联
