# Claw 资讯 🦀

> 2026-04-18 | 数据来源：GitHub OpenClaw Releases

## 今日更新

**openclaw 2026.4.15 正式发布**

### 主要更新

**Anthropic/模型：默认选择升级至 Claude Opus 4.7**

默认 Anthropic 选择升级，添加 `opus` 别名，Claude CLI 默认配置和捆绑图像理解功能全面切换至 Claude Opus 4.7。

**Google/TTS：新增 Gemini 语音合成支持**

为 `google` 插件添加 Gemini 文字转语音支持，包括提供商注册、语音选择、WAV 回复输出、PCM 电话输出以及设置和文档指导。

**控制面板/概览：新增模型认证状态卡片**

新增模型认证状态卡片，显示 OAuth 令牌健康状态和提供商速率限制压力，当 OAuth 令牌即将过期或已过期时发出提醒。背后由新的 `models.authStatus` 网关方法支持，凭证脱敏并缓存 60 秒。

**Memory/LanceDB：新增云存储支持**

`memory-lancedb` 新增云存储支持，持久化记忆索引可在远程对象存储而非仅本地磁盘运行。

**GitHub Copilot/记忆搜索：新增 Copilot 嵌入提供商**

为记忆搜索添加 GitHub Copilot 嵌入提供商，并暴露专用 Copilot 嵌入主机辅助方法，让插件可在远程覆盖、令牌刷新和安全负载验证的条件下复用传输层。

**本地模型代理：新增轻量级工具配置**

新增实验性配置 `agents.defaults.experimental.localModelLean: true`，可在较弱的本地模型配置下丢弃 `browser`、`cron`、`message` 等重量级默认工具，减少提示大小而不改变正常路径。

**插件打包优化：精简发布包体积**

将捆绑插件的运行时依赖本地化到各自所属扩展，裁剪发布文档有效载荷，并加强安装和包管理防护栏。

**Matrix 质量保障：拆分 QA 运行器**

将 Matrix 实时 QA 拆分为源链接的 `qa-matrix` 运行器和仓库私有的 `qa-*` 表面，避免打包和发布构建中包含私有测试代码。

**文档/展示页：全新视觉**

新增可扫描的主横幅、完成部分跳转链接和社区案例响应式视频网格。
