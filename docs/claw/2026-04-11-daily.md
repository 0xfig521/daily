# OpenClaw 资讯 · 2026-04-11

## v2026.4.11-beta.1 + v2026.4.10

**发布于** 2026-04-11

### 核心更新

- Models/Codex: 新增内置 Codex provider 和插件自有 app-server 套件，`codex/gpt-*` 模型使用 Codex 管理的认证、原生线程、模型发现和压缩 (#64298)
- Memory/Active Memory: 新增可选的 Active Memory 插件，为 OpenClaw 在主回复前提供专属记忆子 agent，支持 message/recent/full context 三种模式 (#64368)
- Dreaming/memory-wiki: 新增 ChatGPT 导入功能和「Imported Insights」「Memory Palace」日记子标签 (#64505)
- Control UI/webchat: 将 assistant 的 media/reply/voice 指令渲染为结构化聊天气泡，新增 `[embed ...]` 富输出标签 (#64104)
- Tools/video_generate: 新增 URL-only 模式生成资源交付、typed providerOptions、参考音频输入、`adaptive` 宽高比支持 (#61987, #61988)
