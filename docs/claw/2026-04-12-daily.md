# OpenClaw 资讯 · 2026-04-12

## v2026.4.11

**发布于** 2026-04-12

### 核心更新

- **Dreaming / Memory Wiki** — 新增 ChatGPT 导入功能，新增「Imported Insights」和「Memory Palace」日记子标签，可直接在 UI 中查看导入的源聊天、编译的 wiki 页面和完整源页面 (#64505)
- **Control UI / Webchat** — 将 assistant 的 media/reply/voice 指令渲染为结构化聊天气泡，新增 `[embed ...]` 富输出标签，外部嵌入 URL 需通过配置审核后方可展示 (#64104)
- **Tools / Video Generate** — 新增仅 URL 模式的生成资源交付、typed providerOptions、参考音频输入、per-asset 角色提示、`adaptive` 宽高比支持，以及更高的图像输入上限 (#61987, #61988) Thanks @xieyongliang
- **Feishu** — 改善文档同步与评论功能

## v2026.4.10

**发布于** 2026-04-11

### 核心更新

- **Models / Codex** — 新增内置 Codex provider 和插件自有 app-server 套件，`codex/gpt-*` 模型使用 Codex 管理的认证、原生线程、模型发现和压缩，而 `openai/gpt-*` 继续走常规 OpenAI provider 路径 (#64298)
- **Memory / Active Memory** — 新增可选的 Active Memory 插件，为 OpenClaw 在主回复前提供专属记忆子 agent，支持 `message` / `recent` / `full context` 三种模式配置，包含 `/verbose` 实时检查、提示词/思考过程覆盖，以及可选的转录持久化 (#64368)

## v2026.4.9

**发布于** 2026-04-09

### 核心更新

- **Memory / Dreaming** — 新增基于真实 REM 的回填通道，支持 `rem-harness --path` 历史记录、日记提交/重置流程、更干净的知识提取，以及短期记忆实时晋升机制 (#64298)
- **Control UI / Dreaming** — 新增带时间轴导航的日记视图、回填/重置控件、可追溯的梦境摘要，以及带晋升提示的场景车道 (#64298)
