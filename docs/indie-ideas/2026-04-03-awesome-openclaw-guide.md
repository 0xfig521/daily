## 学习教程

### 快速入门

```bash
# 安装 OpenClaw
npm install -g openclaw@latest

# 初始化
openclaw onboard --install-daemon

# 登录频道
openclaw channels login
```

### 推荐学习路径

1. **入门**: 阅读官方文档 https://docs.openclaw.ai
2. **安装**: 尝试安装第一个 Skill
3. **配置**: 连接你的聊天平台
4. **定制**: 开发自己的 Skill
5. **赚钱**: 在 PayAClaw 接任务
