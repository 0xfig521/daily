# 关于 0xfig

> 探索前沿科技与个体价值

## 🤝 关于项目

**0xfig** 是一个由 AI 驱动的每日资讯聚合平台，专注于为独立开发者提供高质量的信息源和灵感。

## 🎯 我们的使命

让独立开发者每天花最少的时间，获取最有价值的资讯和灵感。

## 📦 每日自动收集

| 分类 | 内容 |
|------|------|
| ☀️ 每日简报 | 5大分类资讯一站式阅读 |
| 💡 独立开发灵感 | 项目创意、赚钱思路 |
| 🤖 AI 创新 | 大模型、AI Agent 最新动态 |
| ⛓️ Web3 前沿 | 区块链、DeFi、NFT |
| 🚀 超级个体 | 一人公司、数字游民 |
| 🦀 Claw 技术 | OpenClaw 生态 |
| 🔥 GitHub Trending | 热门开源项目 |

## 🛠️ 技术栈

- **前端**: VitePress
- **部署**: Cloudflare Pages
- **自动化**: OpenClaw AI Agent
- **收集**: RSS + GitHub API

## 🤖 由 OpenClaw 驱动

0xfig 的每日资讯收集和推送完全由 **OpenClaw** AI Agent 自动完成。
