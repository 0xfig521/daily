## 开发者工具与 AI Agent

### GitHub Stacked PRs — 堆叠式 PR 管理工具

GitHub 官方推出的堆叠式 Pull Request 管理框架，支持将大型 PR 拆分为多个小 PR 并按依赖顺序自动编排。该工具可帮助独立开发者和团队更高效地管理代码审查流程，避免大型 PR 难以 review 的问题。

[查看详情](https://github.github.com/gh-stack/) · 471 分 · 262 评论

---

### GAIA — 本地硬件运行的 AI Agent 开源框架

AMD 推出的开源框架，允许用户在本地硬件上构建和运行 AI Agent。支持消费级 GPU，无需云端 API 即可部署私有化 AI 工作流，适合关注数据隐私的独立开发者。

[查看详情](https://amd-gaia.ai/docs) · 103 分 · 24 评论

---

### N-Day-Bench — LLM 漏洞发现能力基准测试

该基准测试评估前沿 LLM 在真实代码库中发现已知安全漏洞的能力。每月从 GitHub Security Advisories 拉取新案例，防止数据污染导致的作弊问题。目前测试 GPT-5.4、Claude Opus 4.6、Gemini 3.1 Pro 等模型。

[查看详情](https://ndaybench.winfunc.com) · 40 分 · 11 评论

---

### SnapState — AI Agent 工作流的持久化状态管理

专为 AI Agent 工作流设计的持久化状态管理工具，帮助 Agent 在多轮对话和任务切换中保持上下文一致性。适合构建复杂自动化流程的独立开发者。

[查看详情](https://snapstate.dev) · 6 分
